Supported codecs

  • Ogg Vorbis
  • FLAC
  • AAC
  • MP3 (44100, 128bit, 2 channel)
  • WAV
  • AIFF
  • Apple Lossless
  • Codecs supported Mac OS X Core Audio API

Supported tags

  • Vorbis comments / FLAC tags
  • ID3v1.0, 1.1, 2.3+
  • APEv1 and v2

General features

  • M3U and PLS playlist
  • Output device selection
  • Gapless playback
  • Logarithmic volume control
  • Shuffle
  • Seeking
